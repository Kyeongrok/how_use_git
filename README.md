# git 사용법

## remote repository를 만든다
  * git에 가입하고
  * e-mail인증을 하고
  * 로그인을 한 후
  * 프로젝트를 만든다.

## local repository(로컬 리포지토리)만들기
  * mkdir <프로젝트명>
  ex) mkdir how_use_git

  * git init
  * git add README.md
  * git commit -m "first commit"
  * git remote add origin https://github.com/Kyeongrok/how_use_git.git
  * git push -u origin master
